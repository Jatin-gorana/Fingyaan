# FinGyaan

FinGyaan is an interactive and user-friendly platform designed to simplify complex financial concepts and empower users to make informed financial decisions. Through engaging tools like gamification, real-world examples, and personalized recommendations, FinGyaan aims to bridge the financial literacy gap.

---

## Problem Statement

Many individuals, especially beginners, find financial concepts overwhelming. The lack of accessible resources further exacerbates this issue, leading to poor financial decisions. FinGyaan addresses this by offering a platform that:

- Simplifies complex financial topics.
- Engages users through interactive tools.
- Provides personalized insights and recommendations.

---

## Features

- **Interactive Tutorials:** Step-by-step guides on financial concepts.
- **Gamification:** Fun quizzes and challenges to reinforce learning.
- **Real-World Scenarios:** Practical examples for better understanding.
- **Personalized Recommendations:** Insights tailored to user inputs.
- **Dynamic Visualizations:** Modern charts and graphs to analyze finances.

---

## Technology Stack

- **Frontend:** ReactJS, Tailwind CSS
- **Backend:** Firebase (Authentication, Firestore)
- **Charts & Visualization:** Chart.js
- **Hosting:** Firebase Hosting

---

## Installation

Follow these steps to set up the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/FinGyaan.git
   ```
2. Navigate to the project directory:
   ```bash
   cd FinGyaan
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
